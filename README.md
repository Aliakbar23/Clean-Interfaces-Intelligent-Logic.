---

## 🎨 Keunggulan Visual & Fitur Utama

- **Premium Playful Neo-Brutalist Layout:** Menggunakan font *Space Grotesk* dan *Plus Jakarta Sans* dengan skema warna cerah berani (Zero Pink) serta shadow brutalist 3D khas.
- **Aurora Background & Grid Overlay:** Efek ornamen cahaya aurora dinamis yang bergeser perlahan di latar belakang dipadukan dengan overlay grid transparan.
- **Interactive UI Physics Sandbox:** Dilengkapi dengan gelembung mengambang (*floating bubbles*) yang merespons gesekan mouse menggunakan magnetic effects.
- **Interactive Lightbox Gallery:** Galeri sertifikat interaktif berkecepatan tinggi dengan navigasi tombol, pintasan keyboard, serta gestur *swipe* pada layar perangkat mobile.
- **Conversational Blackbox AI CLI Simulator:** Konsol terminal interaktif di mana pengunjung dapat mengetikkan perintah dan berinteraksi layaknya berkirim pesan dengan AI.

## 🕹️ Konsol CLI Terminal & Easter Eggs

Bagian **UI Sandbox** memiliki terminal interaktif. Ketik perintah berikut untuk melihat reaksinya:

| Perintah | Deskripsi |
| :--- | :--- |
| `help` | Menampilkan seluruh daftar perintah asisten virtual. |
| `about` | Informasi singkat profil diri dan fokus keahlian Aliakbar. |
| `skills` | Menampilkan *matrix core technology* (Frontend, Backend, AI/Automation). |
| `projects` | Tautan langsung menuju proyek-proyek live buatan Aliakbar. |
| `certs` | Daftar lengkap sertifikat/prestasi akademik beserta panduan membukanya. |
| `view <no>` | Membuka sertifikat secara layar penuh (contoh: `view 5` untuk Samsung Python). |
| `stats` | **[NEW]** Menampilkan bar visual IPK (Perfect 4.00), SKS, & jumlah proyek per semester. |
| `odyssey` | **[NEW]** Garis waktu (timeline) vertikal interaktif perjalanan semester S1 s.d S6. |
| `hack` | **[NEW]** Masuk ke mode visual *Matrix Digital Rain* (klik layar/ketik apa saja untuk keluar). |
| `secret` | Memicu burst confetti partikel emoji dari dalam konsol terminal. |
| `clear` | Membersihkan layar konsol terminal kembali kosong. |

## 🚀 Cara Menjalankan Secara Lokal

Karena proyek ini murni client-side statis, Anda dapat menjalankannya dengan mudah tanpa perlu melakukan build/compile:

1. Clone repositori ini ke komputer lokal Anda:
   ```bash
   git clone https://github.com/Aliakbar23/Clean-Interfaces-Intelligent-Logic..git
   ```
2. Masuk ke direktori proyek:
   ```bash
   cd Clean-Interfaces-Intelligent-Logic.
   ```
3. Jalankan menggunakan server HTTP lokal pilihan Anda, misalnya dengan `npx http-server`:
   ```bash
   npx http-server -p 8080
   ```
4. Buka browser Anda dan akses `http://localhost:8080`.
